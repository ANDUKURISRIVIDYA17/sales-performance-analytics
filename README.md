## 📌 Project Overview
This project demonstrates an end-to-end data analytics workflow: transforming raw, unformatted e-commerce transaction data into a clean dataset and building an interactive business intelligence dashboard.

The pipeline handles data cleaning programmatically using **Python (Pandas)** and exports structured data to feed a dynamic **Power BI** dashboard, allowing stakeholders to track key revenue metrics, regional performance, and product seasonality.

---

## 🛠️ Tech Stack & Tools
* **Data Engineering & Cleaning:** Python 3.x, Pandas
* **Data Storage:** CSV / Microsoft Excel
* **Business Intelligence & Visualization:** Power BI Desktop
